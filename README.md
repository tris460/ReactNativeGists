# App information

The following screens are implemented in this project:
- A menu working with the Stack navigation
- A fade animation
- A bounce animation
- Switch buttons
- Alert
- Text inputs
- Pull to refresh view
- List divided by sections
- Modal
- Infinite scroll
- Slides
- Dark/Light mode
